Technical Blog - Javascript Fundamentals

When you visit a website or use a web application, three key technologies are working behind the scenes: HTML, CSS, and JavaScript. Each one plays a different role, but they work together to create the websites and apps we use.

HTML – The Structure

HTML (HyperText Markup Language) is the foundation of any web page. Think of it like the frame of a house — it provides the structure. HTML is used to create elements like headings, paragraphs, buttons, links, and images. For example, when you see a title or a button on a website, that’s HTML in action.

Here’s a simple example of HTML code:
<p>Welcome to My Website</h1>
<p>This is an example of a paragraph.</p>

CSS – The Style

CSS (Cascading Style Sheets) is what makes a website look good. It controls the design — things like colors, fonts, layouts, and spacing. If HTML is the frame of the house, CSS is the paint, furniture, and decorations. With CSS, you can make a website look modern, colorful, and easy to navigate.

Example of CSS:
h1 {
color: blue;
font-size: 24px;
}

JavaScript – The Behavior

JavaScript adds interactivity and functionality to a website. It makes things happen — like when you click a button and a pop-up appears, or when content changes without needing to reload the page. If HTML is the frame and CSS is the paint, JavaScript is the electricity that makes everything work.

Example of JavaScript:
document.querySelector('button').onclick = function() {
alert('Button clicked!');
};

How They Work Together

HTML creates the structure, CSS makes it look good, and JavaScript makes it interactive. For example, a "Buy Now" button is created with HTML, styled with CSS to make it look appealing, and JavaScript makes it work when you click it. Together, HTML, CSS, and JavaScript create the seamless web experiences we use every day!
An analogy I would use would be similar to the one used above: the house.

HTML = the structure of the house e.g. the frame

CSS = the style, colour and decorative features of the house

Javascript = the function the utilities of the house e.g. tap turns the water on and off, light switches turn light on and off.

Explain control flow and loops using an example process from everyday

Control flow is the order in which you program an object. For an everyday example, we could say it is like following a method for making a cup of tea:

1 = boil water
2 = if you have tea bags
- Place tea bag in a mug
3 = if you do not have tea bags
- Buy tea bags
- Place tea bag in a mug
3 = pour boiling water in the mug, leaving space for milk
4 = add a splash of milk
5 = drink tea

A loop is when we repeat a task until certain conditions are met.

For example:
Tea is too hot:
- Blow on tea
- Check temperature
- Stop loop until tea is at 60 degrees celsius

Describe what the DOM is and an example of how you might interact with it.

Imagine that the computer screen is a lego house. The DOM (Document Object Model) is like the lego instructions that tell you about the lego bricks in the house. The HTML is the frame of the house, the foundations, door frames etc The CSS is the colour, layout and design. The DOM is what the computer uses when it looks at the instructions of a web page and understands how it is built. We can manipulate the DOM by using javascript which dives into the instructions and pinpoints the exact locations of the ‘bricks’ we want to change in the house.

An example of javascript changing a button on a website would be:
document.getElementById(‘myButton’).innerText = “Hello”

The above syntax finds the button called ‘myButton’ and changes the text within it to say “Hello”

Explain the difference between accessing data from arrays and objects.

ITo put it plainly, an array is like a list of something and an object is like a dictionary for something. An array declares a variable with a list of a certain property. The list always starts an index number starting with 0 as the first item in the list.

An array example:
Fruit:
[0] apple
[1] banana
[2] pear

Or in Javascript:
let fruit = [‘apple’, ‘banana’, ‘pear’]
console.log(fruit[1]) // output is banana

An object declares a variable that has different properties that you access as keys. It acts as a dictionary for the variable that you are declaring.

An object example:
let person = {
name: “Sabrina”,
age: 36,
job: “mum”
}
console.log(person.name) // output is Sabrina

Explain what functions are and why they are helpful.

To put it simply, a function is like a recipe card, it has a set of instructions that make a certain objective happen and you can ‘call’ the function time and time again to see the objective.

A simple example of a function is:
Function sum() {
return a + b;
}

The function is called sum, the inputs (ingredients) are called a and b, the the instructions are to add a and b together.

To use the function it looks like this:
let result = sum(5, 3);
console.log(result) // output is 8
